# dailyPrc

一个持续更新的个人 Python 工具仓库，主要用于日常开发、数据处理、论文出图、医学影像相关实验辅助以及一些维护脚本的整理与归档。

## 项目简介

这个仓库是我在日常学习、科研和项目实践中积累的常用脚本合集。  
内容并不是单一功能的完整框架，而是偏向“实用工具箱”风格，方便随时调用、修改和复用。

当前仓库中的内容主要包括：

- 论文绘图与可视化脚本
- 图像处理与结果展示脚本
- 医学影像相关辅助代码
- 文件处理、格式转换与日常维护脚本
- 按时间或专题整理的实验代码目录

## 适用场景

这个仓库适合以下场景参考：

- 需要快速编写或复用 Python 小工具时
- 需要生成论文插图、三维曲面图、mask 可视化结果时
- 需要处理部分医学影像数据、DICOM/NIfTI 相关任务时
- 需要进行文件整理、PDF 处理、批处理脚本维护时

## 仓库结构说明

当前仓库包含若干按日期命名的目录，以及一些按功能命名的脚本和文件夹。整体更偏向个人工具集合，而不是单一模块化工程。

示例内容包括：

- `viz_qumian.py`：三维曲面可视化示例
- `viz_mask.py`：mask 可视化相关脚本
- `viz_Axial_2d.py`：二维轴向图像展示
- `sam_infer.py`：推理相关脚本
- `changePDF.py`：PDF 处理脚本
- `download_brats2019.py`：数据下载辅助脚本
- `dicom_heart_HX/`：医学影像相关目录
- `utils_all/`：通用工具目录
- `fig/`、`images/`：图片与结果文件目录

> 注：部分目录和脚本来自不同时间段的实验、测试和维护需求，因此命名风格可能不完全统一，后续会逐步整理。
